# react-native-simple-camera

React native simple camera implementation

## Installation

```sh
npm install react-native-simple-camera
```

## Usage

```js
import { SimpleCameraView } from "react-native-simple-camera";

// ...

<SimpleCameraView color="tomato" />
```
